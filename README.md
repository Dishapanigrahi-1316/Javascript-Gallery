# JavaScript Image Gallery

A simple and responsive image gallery project created using HTML, CSS, and JavaScript.

## Features

* Dynamic image loading using JavaScript
* Images stored in an array
* Automatic gallery generation using `forEach()`
* Responsive gallery layout
* Easy to add new images

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

* Image names are stored inside an array
* JavaScript creates `<img>` elements dynamically
* Images are displayed inside the gallery container

## Project Structure

* `index.html`
* `style.css`
* `index.js`
* `image/` folder containing gallery images

## How to Run

1. Download or clone the project
2. Open `index.html`
3. View the image gallery in browser
